HTML

HTML Stands for HyperText Markup Language. It is The Skeleton of a Website.

HTML Tags

We Use tags to define content in HTML.

In this Lecture Notes,we're going to see:

  1. Headings
  2. p tag
  3. Lists
  4. anchor tag
  5. br
  6. hr
  7. Image tag

  1. Headings

    2. To define Header Tags , there are h1 to h8 tags , h1 being given the highest importance and h6 the least.

    We can define these as Follows:

    1. <h1> (content) </h1>

    2. <h2> (content) </h2>

    3. <h3> (content) </h3>

    4. <h4> (content) </h4>

    5. <h5> (content) </h5>

    6. <h6> (content) </h6>

  2. p tag(Paragraph)

    3. This is used to group Sentences Together in the form of a paragraph . Without Paragraph The code looks Dirty.

    The Browsers automatically add a little bit of Line Space Before & after p tag . We should not use Multiple Line Breaks, We Should use Multiple p tags

    <p> (content) </p>

  3. Lists

    4. There are Two Types of Lists:

    1. Ordered List
    2. Unordered List

    li-List Item <li> (item) </li>

    1. Ordered List

      2. This looks Like this: <ol> (content) </ol>

      This creates a Numbering List

      Example: <ol>
      <li> Mango </li>
      <li> Orange </li>
      <li> Banana </li>
      </ol>

    2. Unordered List

      3. This looks Like this: <ul> (content) </ul>

      It Creates new Bullet Points

    Example: <ul>
    <li> Mango </li>
    <li> Orange </li>
    <li> Banana </li>
    </ul>

    Nested List

    Nested Lists looks like this :

    • Frontend

      • HTML
      • CSS
        • onsubmit attribute
        • onclick attribute
      • Javascript

    • Library/Framework

      • ReactJS
        • Hoisting
        • Props

    To create Nested List , we have to add a list inside another list , and it is done as follows:

    <ul>

    <li><h2>Library/Framework</h2> <ul> and add a br after every new line

    <li>ReactJS <ul>
    <li>Hoisting</li>

    </ul></li>
    </ul>
    </ul>

  4. Anchor tag

    5. Anchor tag is used to Link html Page to Another Page

    Is is defined using <a href=(link)> (content) </a>

    Example:
    <a href=ww.google.com>Google </a>

  5. br(Line Break)

    6. This is used to add a line break , After a content , because html doesn't add a new line after each content , we have o use either a line break , or Block Elements , which are elements that always start on a new line.

    br is a self-closing tag

    You can add Muliple Line Breaks to add more space

    This is just used like </br>

  6. hr

    7. This tag is used to add a Horizontal Line To Separate Contents from each Other.

  7. Image tag

    8. this tag is used to add Images

    <image src = (link) alt="">

    You can either give the Link from The local , Or from The Internet

    alt is the text to Show in case the image doesn't get Displayed , the user knows whats the is image about
