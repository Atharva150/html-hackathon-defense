Tables

To create table in html we use the <table> tag.

Inside table tag , we use <tr> to create a new row , and inside new row , we use <td> to insert items into them.

To specify header , we use <th> tag

Example:
<table>
<tr> <td> Mango </td>
<td> Orange </td>
<td> Banana </td>
</tr>

Generally , we use the notations like thead , tbody and tfoot in tables to make code Readable and understand whats going.

thead-Declared in Header , to specify what is the header

tbody-the actual content of the table

tfoot-to add the conclusion

How can we create table like this?

For that we have colspan and rowspan

rowspan is for how many rows we want that cell to take , and colspan is for how many columns that cell should take.

For Example:

Day 9-10 10-11 11-12
Monday Math Science Lab
English History Geography
Tuesday Sports Day

the code for the above is:

<table border="1">
<tr>
<th>Day</th>
<th>9-10</th>
<th>10-11</th>
<th>11-12</th>
</tr>
<tr>
<td rowspan="2">Monday</td>
<td>Math</td>
<td colspan="2">Science Lab</td>
</tr>
<tr>
<td>English</td>
<td>History</td>
<td>Geography</td>
</tr>
<tr>
<td>Tuesday</td>
<td colspan="3">Sports Day</td>
</tr>
</table>
